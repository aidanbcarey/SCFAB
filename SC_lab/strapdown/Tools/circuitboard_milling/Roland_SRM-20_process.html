Cutting Circuitboards on the Roland SRM-20 - basic process

Run modserver (if it is not running already).

  1. On the Windows Desktop, double-click the shortcut "start mods"
  2. A command window will appear, with the text "listening for connection from client address 127.0.0.1 on server port 1234"

Open fab mods.

  1. Open the webpage on the desktop to run the local mods program: "PCB_Harvard_SRM20"
  2. You should see "socket opened" in the WebSocket device window.
  3. Machine should respond to the "move to origin" and "move to home" keys.

Place a circuitboard blank on the sacrificial layer.

  1. Clean both circuitboard back and sac. layer.
  2. Cover the back of the blank double-stick tape, no overlaps or creases.

Insert the tool and zero it.

  1. Hit "move to home" to bring spindle up.
  2. Put 1/64" bit in collet (insert high - almost all the shaft in the collet)
  3. Hit "move to origin". Adjust x,y,and z to as needed.
  4. Loosen the toolbit screw, let the tool rest on the board and then tighten its screw.

Run traces file.

  1. Select png file for traces.
  2. Select defaults for traces.
  3. Calculate.
  4. Send file to device.

Run outline file.

  1. Change to 1/32" bit. Leave bit high in collet so that it does not crash into bed when zeroing.
  2. Go to the origin. As before, move the tool to the board surface to set the z-zero.
  3. Select defaults for outline
  4. Calculate
  5. Send file to device.

When the machine finishes running, wait ~5 seconds before opening shield. If you open too soon, communication is lost. In that case, turn off SRM20 and turn it on again. Communication should re-establish.
