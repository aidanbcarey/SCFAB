Cutting Circuitboards on the Shopbot - basic process

Safety Information

  • Training checklist
  • Machine-specific safety concerns.
    1. Before running the spindle, put shield in place.
    2. Ensure correct z-zero so that tools do not break due to too-deep cuts.
  • Using fab modules
    1. Load settings file
    2. Load png file
    3. Choose output format "ShopBot" (sbp)
    4. Choose process for PCB traces
    5. Check parameters, calculate, and save

  • Running the Shopbot
    1. Remove the shield.
    2. Attach the board to appropriate surface with double tape.
    3. Turn on Red switch and remove key/wrench from spindle switch.
    4. Open the ShopBot software.
    5. Raise head and insert 1/64" tool bit(insert high - almost all the shaft in the collet.)
    6. With hands out of the way, open yellow pad and define zero in x, y, and z, using the blue "zero axes" button.
    7. Move the spindle head to the middle of the board, where you will zero the z-axis.
    8. Loosen tool bit and move it to the board, holding it down as you finger-tighten the collet.
    9. Tighten the collet with the wrenches.
    10. Using the yellow keypad, carefully reduce the gap between toolbit and surface to zero, using shims.
    11. Define zero in z, using the blue "zero axes" button.
    12. Close the yellow keypad.
    13. Make sure spindle control is on and set for maximum speed.
    14. Put the shield in place
    15. Run the file that you generated in fab modules, using the green "Cut Part" button and following the dialog boxes.
    16. When the job completes, use yellow keypad to move the head up and out. Vacuum dust and check traces.
    17. Change to the 1/32" bit for cutting the outline, again inserting it far into the collet.
    18. Zero the z, as in 6-10 above
    19. Replace the shield.
    20. Close the yellow keypad and run the outline file.
    21. When job is done, use the yellow keypad to move the head out of the way, remove the shield, and vacuum up the dust.
