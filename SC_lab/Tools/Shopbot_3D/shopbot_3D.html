Shopbot Procedures - Desktop Shopbot.

Safety Guidelines

  1. Wear eye and ear protection.
  2. Closed-toe shoes.
  3. No loose hair or clothing.
  4. Never reach into area with moving tools.
  5. Know how to stop machine before you start it.
  6. Never work alone or when tired.

Creating 3D Toolpaths in VCarve. (Shopbot-provided software)

  1. Measure material
  2. Open software, choose “new file”.
  3. Material setup
  4. Using "Drawing" tab, import stl file.
  5. Define position of model in material.
  6. Create roughing toolpath (following are suggestions for wax).
    • tool - 1/8”
    • spindle speed 18000 rpm
    • 2 in/sec
    • cut depth per pass 0.125”
    • stepover 50%.
  7. Create finishing toolpath
    • Choose tool.
    • spindle speed 18000 rpm
    • Decrease stepover as needed.
    • If needed, make additional pass at 90 degrees to first pass.
  8. Save toolpaths.

Running a toolpath file on the Shopbot Desktop Machine.

  1. Attach material to bed.
  2. Insert the appropriate bit.
  3. Zero the bit in x, y, and z.
  4. Make sure spindle is turned on and speed set as desired (speed is in rev. per second)
  5. Do an air cut.
    • move z-zero up.
    • run file
  6. Cut part
    • Return z-zero to real zero. Check.
    • run file
    • Spindle turns on when you press "OK" - make sure spindle control on shopbot desktop is on.
