Procedure for turning a sketch into a vector file.

  • Import bitmap of sketch.
  • Path -> Trace Bitmap. Open dialog box and hit "OK".
  • Select and isolate vector paths.
  • remove fill color, add line color
  • Break apart
  • Smooth. Export as dxf file.
