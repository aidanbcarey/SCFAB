Cutting Circuitboards on the Roland MDX-20 (Modela)- basic process

Safety Information

  • Training checklist
  • Machine-specific safety concerns.
    1. Before running the spindle, make sure hands are out of the workspace and shield is in place.
    2. Ensure correct z-zero so that tools do not break due to too-deep cuts.

Running mods server (if it is not running already).

  1. Open a terminal window
  2. type: "start_mods_server"
  3. The program should respond, "listening for ..... port 1234"
  4. If you get an error message indicating that a process is already going on, go to the Mac Activity Monitor, search for a process containing the name "Node", and terminate it. Then repeat the "start_mods_server" command.

Using fab modules

  1. Open fab modules by double-clicking the icon on the desktop for the file Modela_PCB.html.
  2. You should see "serial port opened" in the WebSocket print window.
  3. If necessary, hit "move home and stop" to bring spindle up and table forward.

Place a circuitboard blank on the sacrificial layer.

  1. Clean both circuitboard back and sac. layer.
  2. Cover the back of the blank double-stick tape, no overlaps or creases.
  3. Press down on table firmly, assuring no gaps and no wiggle.

The VIEW button toggles the machine between a "view" mode, in which the bed is forward and the spindle raised (view light on) and the mode in which the machine is receptive to serial commands (view light off).

Insert the tool and zero it.

  1. Put 1/64" bit in collet (insert high - almost all the shaft in the collet)
  2. Hit "move to origin".
  3. Use up and down keys on Modela so that spindle is within 5mm of its minimum "z" position.
  4. Adjust x and y as needed.
  5. Loosen the toolbit screw, let the tool rest on the board and then tighten its screw.

Run traces file.

  1. Select png file for traces
  2. Select defaults for traces
  3. Calculate.
  4. Send file.

Run outline file.

  1. Change to 1/32" bit. Leave bit high in collet so that it does not crash into bed when zeroing.
  2. Go to the origin. As before, move the tool to the board surface to set the z-zero.
  3. Select png file for outline.
  4. Select defaults for outline.
  5. Calculate.
  6. Send file.

To Cancel

  1. hit "VIEW"
  2. Flush the Modela's data buffer by pressing "down" and "up" simultaneously while in the VIEW mode.
