Shopbot Procedures

Safety Guidelines

  1. Wear eye and ear protection.
  2. Closed-toe shoes.
  3. No loose hair or clothing.
  4. Never reach into area with moving tools.
  5. Know how to stop machine before you start it.
  6. Never work alone or when tired.

Creating a Toolpath in VCarve. (Shopbot-provided software)

  1. Measure material
  2. Open software, choose “new file”.
  3. Material setup
  4. Import vector or draw vector.
  5. Create toolpath (following are suggestions for OSB).
    • tool - 1/8”
    • spindle speed 18000 rpm
    • 3 in/sec
    • cut depth per pass 0.2”
    • tabs can be 0.5” wide by 0.1” thick.
  6. Save toolpath(s).

Running a shopbot toolpath file.

  1. Attach material to bed.
  2. Insert the appropriate bit.
  3. Zero the bit in x, y, and z.
  4. Make sure spindle control window is open (Tools menu)
  5. Do an air cut.
    • move z-zero up.
    • run file
    • turn spindle on.
  6. Cut part
    • Return z-zero to real zero. Check.
    • run file
    • Turn dust collector on
    • turn spindle on
