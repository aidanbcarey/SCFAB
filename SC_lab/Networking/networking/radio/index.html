[Return to PS70 Networking menu](../index.html) ###Step-by-step instructions for radio connection. <b>1. Install library for this radio through the Arduino Libary Manager: "RF24 by TMRh20"</b> <b>2. Connect female/male wires to radio chip as shown below. </b> <b>3. Connect wires to microcontroller pins, following the table below, or the instructions [here](http://tmrh20.github.io/RF24/) </b> <b>4. Load the appropriate send and receive sketches. Check for success using the Arduino Serial Monitor. </b> ###Pin connection for radios ![pic](radio_pinout.jpg) ![pic](radio_pic.jpg) ###Pin connections to microcontroller Pin on radio |D11 board |ItsyBitsy D51 board ------------- | ------------- 1 GND| GND|GND 2 VCC| 3V3|3V3 3 CE| pin 8|pin 7 4 CS or SS| pin 15 (=SS)|pin 9 5 SCK| pin 5|SCK 6 MOSI| pin 4|MO (=MOSI) 7 MISO| pin 14|MI (=MISO) [For other boards, follow connection instructions](http://tmrh20.github.io/RF24/) from Github site for the RF24 library. ![pic](d11_conn.jpg) ![pic](itsy_conn.jpg) ###Code [Transmit code ](./simple_xmit_Itsy.txt). For ItsyBitsy M4 board. Change CE and CS pins for use with other boards. [Receive code](./simple_rec_SAMD11.txt). Pin numbers appropriate for ATDSAMD11 M0 board. Change for other boards. ###Notes For SAMD11 Mattair board, Set serial config to ONE_UART_NO_WIRE_ONE_SPI.
