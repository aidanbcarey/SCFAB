<br> ##Adafruit boards for use in PS70 Spring 2020. ###Adafruit ItsbyBitsy M4 Express This board uses the 32 bit ATSAMD51 microcontroller, and adds 2MB of flash memory. It is intended as the main microcontoller board for PS 70. ![itsy](./itsy.png) - [website](https://www.adafruit.com/product/3800) and learning guide. - Things to try when the board does not appear. - double click the reset button. - quit and restart Arduino. - Check Arduino version??? - When all else fails, reload the bootloader. ###Adafruit Feather M4 Express. Same microcontroller, with a few more pins. Can do I2S for use with audio libraries. ![feather](./feather.png) ![feather_pinout](./Feather_M4_pinout.png) - [website.](https://learn.adafruit.com/adafruit-feather-m4-express-atsamd51)
